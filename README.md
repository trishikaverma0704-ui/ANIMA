## Problem
Pet parents lack a single trusted platform for community help, local alerts, and emergencies.

## Solution
PawTown is a digital town square where pet parents connect through community feeds,
neighbourhood circles, and an AI agent that provides guidance and emergency support.

## Core Features
- Community Feed
- Neighbourhood Circles (location-based)
- AI Pet Care Agent
- Pet Profiles
- Emergency Broadcast Mode
- Rescue & NGO Connect (future scope)

## How It Works
1. User creates a pet profile
2. Posts a query or emergency
3. AI agent classifies and responds
4. Nearby users receive alerts
